Hospital Management System

Hospital Management System is a desktop-based application developed using Java, JavaFX, JDBC, Oracle Database, and PL/SQL. The project is designed to manage core hospital workflows such as patient registration, doctor management, appointment scheduling, medical visits, prescriptions, lab test orders, medicine inventory, billing, and report generation.

The system follows an MVC-based architecture, separating the user interface, business logic, database access, and data models for better maintainability. It uses Oracle Database as the backend with relational tables, constraints, sequences, triggers, and stored procedures to ensure data consistency and automate important hospital operations.

Key Features
Patient registration and management
Doctor and department management
Appointment booking with duplicate slot prevention
Visit records with diagnosis, notes, and follow-up details
Prescription management with medicine allocation
Automatic medicine stock update after prescription
Lab test ordering and result tracking
Billing system with consultation, medicine, and lab charges
Payment tracking with automatic bill status update
Dashboard and reports for hospital data overview
Tech Stack

Frontend: JavaFX, FXML, CSS
Backend: Java, JDBC
Database: Oracle Database, SQL, PL/SQL
Build Tool: Maven
Architecture: MVC Pattern
